## ðŸ› Troubleshooting

### "Model file not found"
**Problem:** Running `app.py` or `predict.py` without training first.  
**Solution:** Run `python train.py` to generate the model file.

### "train.csv not found"
**Problem:** Dataset not downloaded.  
**Solution:** Run `python download_data.py` or download manually from Kaggle.

### Import errors
**Problem:** Missing dependencies.  
**Solution:** 
```bash
pip install -r requirements.txt
```

### Kaggle API errors
**Problem:** Kaggle credentials not configured.  
**Solution:** Follow instructions in `download_data.py` or download files manually.

## ðŸ’¡ Tips

1. **First time setup:**  
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # macOS/Linux
   pip install -r requirements.txt
   python download_data.py
   python train.py
   streamlit run app.py
   ```

2. **Faster training:** Edit `train.py` and change `sample_size=500000` to a smaller number (e.g., 100000).

3. **Testing without full dataset:** Create a small sample:
   ```bash
   head -n 10000 data/raw/train.csv > data/raw/train_small.csv
   # Then edit train.py to use train_small.csv
   ```

4. **Model already exists:** The repo includes a pre-trained `taxi_fare_pipeline.pkl` so you can run the app immediately without training.
